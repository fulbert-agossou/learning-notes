# Notes — CompTIA Network+

## Concepts de base

- **LAN** : Local Area Network — réseau local (ex: le WiFi à la maison)
- **WAN** : Wide Area Network — réseau étendu (ex: Internet)
- **IP** : Adresse qui identifie un appareil sur un réseau
